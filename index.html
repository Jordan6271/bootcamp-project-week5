Escape From The Manor


Rules of the Game


Goal

The aim of the game is to find your way safely out of the manor. There will be several obstacles in your way to freedom, some more challenging than others. You must investigate the rooms of the manor to find helpful hints and tools which you can use to make it easier to succeed in any trials you might face. There are few guaranteed victories, and if you choose to begin a fight unprepared, you may fail, and must restart the game to try again.

Controls

In any given room, there may be objects that you can interact with. To interact with an item, simply type the name of that item. For example, if a room was described to have a desk with a drawer, you should either type the word "desk" or "drawer" to interact with the object. Do not type "open drawer" or "search desk".

If you try to interact with an object that does not have any interactions associated with it, the error message "You can't do that." will display.

The directions you can move will be indicated within the description of each room. To move to a new room, type "north", "east", "south", or "west". If you type a direction that is not applicable to your current room, the error message "You can't go that way." will display.

Commands that you type are not case sensitive.

Game Play

When interacting with an object, you may receive an item. A message will be displayed in blue confirming an item has either been used immediately or placed in your inventory. Your inventory can be viewed in the bottom right of the screen.

Two items may interact with one another. If you type the name of an item in your inventory, and you are in the room with an object that can be combined with it, you will receive a new item, and a message will be displayed to this effect. This also works if you type the name of an object in a room which is associated with one in your inventory.

If an item in your inventory can be used during a fight, this will automatically occur.

Your health will increase or decrease depending on certain interactions you make, and your performance in different scenarios. This will be indicated by green text if you gain health, or red text if you lose health. If you perform poorly and start a fight unprepared, you may fail. In this case, you must restart the game to try again.

Good Luck!
